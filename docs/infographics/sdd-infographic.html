🛡️ Safety Features

Circuit Breaker
Warnings at 25, 40 iterations. Prevents runaway loops.
Catastrophic Filter
Blocks: rm -rf /, chmod 777 /, dd to /dev/sd*
Hook Chain
Two-stage validation: block-dangerous-git → block-catastrophic-commands

🚪 Work Gates (.autogate.json)

ready
Boolean. Allow autonomous work on item.
stop_at_phase
Integer | null. Halt after phase N for review.
Integrates with Stop hook. Fail-safe: errors allow work to continue.

🔁 Loop Controller (sdd-loop.sh)

Multi-repo execution — Run across multiple repos autonomously
Configurable limits — Set iteration caps, timeouts, resource constraints
Progress tracking — Monitors state, prevents infinite loops
Graceful shutdown — Respects gates, archives work safely
