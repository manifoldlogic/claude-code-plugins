# Analysis Plugin

Deep analytical thinking and problem-solving tools for Claude Code.

## Overview

The Analysis plugin provides multi-dimensional analysis capabilities for complex technical and strategic decisions. It helps break down complex problems, evaluate multiple perspectives, and generate comprehensive recommendations.

## Commands

### `/ultrathink`

Deep analysis and problem solving with multi-dimensional thinking.

**Usage:**
```bash
/ultrathink [problem or question to analyze]
```

**Examples:**
```bash
# Architectural decision
/ultrathink Should we migrate to microservices or improve our monolith?

# Complex problem solving
/ultrathink How do we scale our system to handle 10x traffic while reducing costs?

# Strategic planning
/ultrathink What technology stack should we choose for our next-gen platform?

# Design challenge
/ultrathink How can we improve our API to be more developer-friendly while maintaining backward compatibility?
```

**Features:**
- Multi-dimensional analysis (Technical, Business, User, System perspectives)
- Multiple solution generation with trade-off analysis
- First principles and systems thinking
- Cross-domain insights
- Structured recommendations with risk assessment

## Installation

This plugin is part of the claude-code-plugins repository and can be installed via the Claude Code plugin system.
